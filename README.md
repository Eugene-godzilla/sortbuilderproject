# Sorting Application (InsertionSort + Builder, Strategy patterns)

## Описание проекта
Приложение реализует **сортировку пользовательских объектов** (на примере класса `Car`) с использованием **алгоритма вставками (Insertion Sort)**.  
Программа демонстрирует применение паттернов **Strategy** (для выбора критерия сортировки) и **Builder** (для построения объектов).  

Приложение позволяет пользователю:
- выбрать способ заполнения данных (вручную, из файла или случайно);
- выбрать критерий сортировки (по модели, мощности или году выпуска);
- просмотреть и сохранить результаты сортировки.

---

## Архитектура проекта
```
com.project
│
├── model/
│ └── Car.java # Класс модели + Builder
│
├── io/
│ ├── InputManager.java # Ввод данных (ручной, из файла, случайный)
│ └── OutputManager.java # Вывод и сохранение данных
│
├── sort/
│ └── InsertionSort.java # Алгоритм сортировки вставками
│
├── strategy/
│ ├── CarComparatorStrategy.java # Интерфейс стратегии (расширяет Comparator)
│ ├── ModelComparator.java # Сортировка по модели
│ ├── PowerComparator.java # Сортировка по мощности
│ ├── YearComparator.java # Сортировка по году
│ └── ContextSort.java # Контекст, управляющий выбором стратегии
│
├── mainapp/
│ └── Application.java # Логика меню и взаимодействия с пользователем
│
└── Main.java # Точка входа
```
